# RTKLIB-Manual-CN

## 项目初衷

RTKLIB提供的参考手册组织方式过于冗杂，至少我个人认为它谈不上特别有条理。项目维护在线文档：[RTKLIB手册中文版](https://salmoshu.github.io/algorithm/RTKLIB-Manual-CN/01-overview.html)

## 项目目标

翻译并重新组织RTKLIB手册，提供算法原理或软件使用进一步的解释说明，以下为当前进展:

- [x] 基础翻译
- [x] 内容的重新组织
- [ ] 算法附录的解释说明
- [ ] 算法源码解析
